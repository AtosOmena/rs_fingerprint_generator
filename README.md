# RS Fingerprint Generator

Este repositório foi idealizado para ser um módulodo para o repositório ...

Este repositório contém a implementação em Rust de uma coleta de dados de fingerprint utilizando a biblioteca fingerpint_rs a fim de ser compilada em WASM.

Para utilizá-lo é necessário ter o compilador de `Rust` e o gerenciador de pacotes `Cargo` instaldo.

Para instalar todas as dependencias do projeto utilize o comando:
```bash
cargo build
```

Depois basta seguir as instruções no repositório ...
